#  Business Sales Decision Dashboard

##  Overview
This project presents an interactive Power BI dashboard designed to analyze sales and profit performance across regions and products. It also incorporates a rule-based decision system to provide actionable business recommendations.



##  Key Features
- KPI Cards displaying Total Sales and Total Profit
- Daily Sales Trend Analysis (Time Series)
- Profit Comparison by Region (Color-coded performance)
- Product-level filtering using slicers
- Decision system suggesting:
  - Increase Marketing
  - Reduce Cost
  - All Good



##  Key Insights
- West region generates the highest profit.
- East region shows comparatively lower performance.
- Sales trends indicate fluctuations, suggesting inconsistent demand.
- Strategic improvements in low-performing regions can increase overall profitability.



##  Business Value
This dashboard helps stakeholders:
- Identify high and low-performing regions
- Make data-driven marketing and cost decisions
- Monitor sales trends over time
- Take actionable steps based on performance
